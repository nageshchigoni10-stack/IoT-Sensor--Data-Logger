# IoT Sensor Data Logger (C++ Project)

This project simulates IoT temperature and humidity sensors.
Data is logged to a CSV file and analyzed for average, max, and min readings.

## Features
- Generates random temperature & humidity data
- Logs readings with timestamps
- Analyzes stored data
- Menu-driven console interface
